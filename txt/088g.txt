Davidův potomku slavný
1. Davidův potomku slavný, / světlo patriarchů dávných, / ó svatý Josefe, / snoubenče Boží Rodičky, / pros za nás nyní i vždycky, / pěstoune Ježíšův.
2. Čisté Panny cudný strážce, / Syna Božího ochránce, / ó svatý Josefe, / přimluv se za nás zvlášť nyní, / ty hlavo svaté Rodiny, / živiteli Kristův.
3. Muži Bohem vyvolený, / spravedlivý, čistý, věrný, / ó svatý Josefe, / zrcadlo trpělivosti, / k moudrosti a statečnosti / uč nás jít cestou tvou.
4. Kéž jak ty ctíme chudobu, / věrní jsme Božímu slovu, / ó svatý Josefe; / tys zářným vzorem otcům všem / a dělníkům jsi příkladem, / buď rodin oporou.
5. Veď naše srdce k čistotě, / utěšuj v těžkém životě, / ó svatý Josefe, / vlévej naději nemocným, / buď v smrti ochráncem mocným, / vypros v nebi místa.
6. Pomoz nám všem žíti svatě, / záštitou buď církve svaté, / ó svatý Josefe, / ducha zla od nás zapuzuj, / jednou u soudu při nás stůj, / přimluv se u Krista.
