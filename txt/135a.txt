Svatý Václave, vévodo české země [starší]
1. Svatý Václave, / vévodo české země, / kníže náš, / pros za nás Boha, / svatého Ducha. / Kyrieleison.
2. Nebeské je dvorstvo krásné, / blaze tomu, kdo tam dojde, / v život věčný, / oheň jasný / svatého Ducha. / Kyrieleison.
3. Pomoci my tvé žádáme, / smiluj se nad námi, / utěš smutné, / odžeň vše zlé, / svatý Václave. / Kyrieleison.
4. Ty jsi dědic české země, / rozpomeň se na své plémě, / nedej zahynouti / nám ni budoucím, / svatý Václave. / Kriste, eleison.
5. Maria, Matko žádoucí, / tys královna všemohoucí, / prosiž za nás, / za křesťany, / svého Syna. / Kriste, eleison.
6. Andělé svatí, nebeští, / račte nás k sobě přivésti, / tam, kde chvála / nepřestává / věčného Boha. / Kriste, eleison.
