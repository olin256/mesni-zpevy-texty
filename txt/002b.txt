Jakožto růžička krásná
1. Jakožto růžička krásná / a jako hvězda přejasná, / Matka Krále nebeského / vzešla z rodu královského, / Panna čistá.
2. Adamův dluh je zaplacen, / člověk životu navrácen / skrz plod té Panny přečisté, / přišlo nám spasení jisté / skrze Krista.
3. Naše přijal člověčenství, / vrátil Boží společenství / lidu hříchem ztrávenému, / ďábelskou lstí raněnému / přesmrtelně.
4. Vykoupil nás sám krví svou, / zvítězil nad smrtí věčnou, / obveselil vězně smutné, / vyvedl z vazby ukrutné / přeradostně.
5. Kriste, Synu Panny čisté, / naše útočiště jisté, / rač ovce stáda malého / pod korouhev kříže svého / shromážditi.
6. Pokorně, Pane, prosíme: / Život, ve který věříme, / v den, kdy přijdeš, rač nám dáti, / s Matkou tvou se radovati, / tě chváliti.
