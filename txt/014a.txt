Raduj se všechno stvoření
1. Raduj se všechno stvoření, / radost nová, / z Krista Pána narození, / radost neslýchaná.
2. Jasnost blesku nebeského, / radost nová, / posel z života čistého, / radost neslýchaná.
3. Vydala vláhu nebesa, / radost nová, / sprchla z nebe na zem rosa, / radost neslýchaná.
4. Spravedlivý z nebe sstoupil, / radost nová, / Kristus, aby nás vykoupil, / radost neslýchaná.
5. Co zvěstovali proroci, / radost nová, / zjevilo se této noci, /  radost neslýchaná.
6. Andělé z nebe zpívají, / radost nová, / radost na zemi zvěstují, / radost neslýchaná.
7. Ze se narodil Spasitel, / radost nová, / nás všech hříšných Vykupitel, / radost neslýchaná.
8. Jdou pastýři, nemeškají, / radost nová, / dítě to naleznou v stáji, / radost neslýchaná.
9. Vidí matičku s synáčkem, / radost nová, / v nouzi být s krásným robátkem, / radost neslýchaná.
10. Diví se, však tomu věří, / radost nová, / že Syn Boží v jeslích leží, / radost neslýchaná.
12. Hvězda od východu slunce, / radost nová, / do Betléma vede krále, / radost neslýchaná.
16. Tam chvátají, hvězdu vidí, / radost nová, / která jim na cestě svítí, / radost neslýchaná.
18. Králové dary skládají, / radost nová, / ochotně se k poctě mají, / radost neslýchaná.
22. Ježíši, v chlévě ležící, / radost nová, / hvězdou jsi v nebi blyštící, / radost neslýchaná.
23. Zaskvěj se v zatmělých srdcích, / radost nová, / tebe vyhledávajících, / radost neslýchaná.
24. Dej, bychom v tebe věřili, / radost nová, / s tebou věčně kralovali, / radost neslýchaná.
