Raduj se všechno stvoření
Raduj se všechno stvoření, / radost nová, / z Krista Pána narození, / radost neslýchaná.
Jasnost blesku nebeského, / radost nová, / posel z života čistého, / radost neslýchaná.
Vydala vláhu nebesa, / radost nová, / sprchla z nebe na zem rosa, / radost neslýchaná.
Spravedlivý z nebe sstoupil, / radost nová, / Kristus, aby nás vykoupil, / radost neslýchaná.
Co zvěstovali proroci, / radost nová, / zjevilo se této noci, /  radost neslýchaná.
Andělé z nebe zpívají, / radost nová, / radost na zemi zvěstují, / radost neslýchaná.
Ze se narodil Spasitel, / radost nová, / nás všech hříšných Vykupitel, / radost neslýchaná.
Jdou pastýři, nemeškají, / radost nová, / dítě to naleznou v stáji, / radost neslýchaná.
Vidí matičku s synáčkem, / radost nová, / v nouzi být s krásným robátkem, / radost neslýchaná.
Diví se, však tomu věří, / radost nová, / že Syn Boží v jeslích leží, / radost neslýchaná.
Hvězda od východu slunce, / radost nová, / do Betléma vede krále, / radost neslýchaná.
Tam chvátají, hvězdu vidí, / radost nová, / která jim na cestě svítí, / radost neslýchaná.
Králové dary skládají, / radost nová, / ochotně se k poctě mají, / radost neslýchaná.
Ježíši, v chlévě ležící, / radost nová, / hvězdou jsi v nebi blyštící, / radost neslýchaná.
Zaskvěj se v zatmělých srdcích, / radost nová, / tebe vyhledávajících, / radost neslýchaná.
Dej, bychom v tebe věřili, / radost nová, / s tebou věčně kralovali, / radost neslýchaná.
