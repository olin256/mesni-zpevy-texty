Ó Beránku nevinný, Pane Jezu Kriste
1. Ó Beránku nevinný, / Pane Jezu Kriste, / odpusť naše mnohé viny, / Synu Panny čisté; / a smiluj se nad námi.
2. Víme, že jsme zhřešili / proti tvé milosti, / přikázání tvá rušili / svými nepravostmi; / ó smiluj se nad námi.
4. Ty jsi Beránek tichý / a Syn Boží pravý, / který snímá světa hříchy, / jak nám Písmo praví; / ó smiluj se nad námi.
5. Tys ta jediná oběť / dostiučinění, / tys výplata za všechen svět, / tys naše spasení; / ó smiluj se nad námi.
6. Ty jsi naše naděje, / útočiště jisté, / bez tebe se nám zle děje, / bídní jsme zajisté; / ó smiluj se nad námi.
8. Tys ten lékař milostný, / jenž sám léčil cele / ty, jež ranil ďábel zlostný / na duši, na těle; / ó smiluj se nad námi.
9. Proto voláme k tobě, / my bídní hříšníci, / neboť ty nás všechny k sobě / přijímáš kající; / ó smiluj se nad námi.
10. Před Otcem nás omlouvej, / ukaž mu své rány, / za nás hříšné se přimlouvej, / ukaž bok proklaný; / ó smiluj se nad námi.
12. Marii Magdalénu / přijal jsi na milost, / hříšnou, kající však ženu. / Dej též nám svou milost / a smiluj se nad námi.
13. Na lotra kajícího / rozpomenul jsi se, / do ráje uvedl jsi ho, / na nás rozpomeň se; / a smiluj se nad námi.
14. Přijals Petra a Pavla, / i jiné hříšníky, / neodlučuj nás od sebe, / nyní i navěky; / a smiluj se nad námi.
