Slyšme z úst Pána našeho
1. Slyšme z úst Pána našeho, / co nám schvaluje dobrého, / jak nás vede k účastenství / pravého blahoslavení.
2. Pán učí: Blahoslavení / chudí duchem, v kterých není / pýcha ni žádost bohatství, / těch je nebeské království.
3. Blahoslavení lkající, / rozličný kříž snášející, / nechají světa marností, / budou potěšeni dosti.
4. Blahoslavení jsou tiší, / kteří slyšíce neslyší, / v křivdách o pomstu nestojí, / na zemi nejspíš obstojí.
5. Blahoslavení jsou věčně, / kdo spravedlnost srdečně / milují a lační po ní, / jistě budou nasycení.
6. Jsou také blahoslavení / všichni lidé milosrdní, / neb milosrdenství dojdou, / bědám časným, věčným ujdou.
7. Blahoslavenství mnohého / jsou všichni srdce čistého, / neboť každý má věděti, / že budou Boha viděti.
8. Blahoslavení také jsou / ti, kdo sváry těžce nesou, / bližní smiřují, rovnají, / synů Božích jméno mají.
9. Také jsou blahoslavení, / kdo jsou pro pravdu věznění, / vyhnanství neb smrt snášejí, / ti do nebe náležejí.
10. Vpravdě jste blahoslavení, / jestliže by vám lidé zlí / zlořečili, trápíce vás, / a mluvili vše zlé o vás.
11. Radujte se z té milosti, / že je v nebeské radosti / složena hojná odplata, / která vám nebude vzata.
