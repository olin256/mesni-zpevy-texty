Svaté Trojici jasné
1. Svaté Trojici jasné. / Marii Panně slavné, / zpívejme svatým, / všem zástupům nebeským, / svatým andělům, / patriarchům, prorokům, / všem svatým apoštolům.
2. Všem svatým mučedníkům, / vyznavačům, panicům / i svatým pannám, / vdovám, svatým manželkám, / všem vyvoJeným, / na nebi kralujícím, / na tvář Boží patřícím.
3. Svatým Božím zpíváme, / na přímluvu čekáme, / chválu vzdáváme, / celým srdcem plesáme / s velkou radostí, / žádajíce milosti / a nebeské jasnosti.
4. Ty, Králi všemohoucí, / Spasiteli žádoucí, / přemilostivý, / pro svou Matku přesvatou, / a požehnanou, / pamatuj na nás hříšné, / smiluj se pro své svaté.
5. Jak je blaze všem svatým, / chudým, tichým, kajícím / i spravedlivým, / milosrdným a čistým / i všem pokorným, / kteří mnoho trpěli, / aby Boha viděli.
7. Jako sám Kristus trpěl / a mnohé úzkosti měl, / tak jeho svatí / za ním také kráčeli, / pro něj trpěli; / Ježíš je posilňoval, / korunu jim za to dal.
