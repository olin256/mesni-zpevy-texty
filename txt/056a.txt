Kristus, jenž byl na smrt dán
1. Kristus, jenž byl na smrt dán / pro naše provinění, / slavně zase z mrtvých vstal / nám k ospravedlnění. / Z toho my se radujme / a Boha oslavujme, / zpívejme aleluja, / aleluja.
3. Ježíš Kristus, Boží Syn, / přišel na místo naše, / vykonal přeslavný čin, / vzal všechny hříchy na se, / shladil je svou obětí, / a tak odňal moc smrti; / již svůj osten ztratila, / aleluja.
4. Divný boj, když se smrtí / život těžce zápasil, / život zdál se klesati, / však na konec zvítězil, / potřel smrt ve vítězství / dle prorockých svědectví, / posměch z ní již učiněn, / aleluja.
5. On je Beránek pravý, / o němž Bůh zdávna hlásal, / pomazat dveře srdcí / jeho krví přikázal. / Kdo to s vírou učiní, / anděl zhoubce jej míjí, / přístup k němu nemaje, / aleluja.
6. Slavme tedy skvoucí den, / který nám učinil Pán, / svítí nám přejasně v něm / ten, který je slunce sám, / Slunce spravedlnosti / a paprsek věčnosti; / noc hříchu pominula, / aleluja.
7. Společně oslavujme / již hody Beránkovy, / kvas starý odstraňujme, / máme být lidé noví. / Kristus je stravou naší, / krmící sladce duši; / víra nežádá více, / aleluja.
