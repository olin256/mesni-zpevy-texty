Zachovej nás při svém slovu
1. Zachovej nás při svém slovu / a dej k němu víru živou, / ty sám, Ježíši, Pane náš, / neboť ty mdlobu nás všech znáš.
2. Ó dobrý Pastýři, Kriste, / opatruj malé stádce své, / věrnými správci obdař nás, / nájemců nedbalých zbav nás.
3. Poněvadž sami ze sebe / nemůžeme nic bez tebe, / v tvou ochranu se dáváme / a za to tebe žádáme:
4. Pas nás na horách zelených, / na trávnících přerozkošných, / pastvu slova svého daruj, / bludů škodlivých uvaruj.
5. Ať moc pravdy slova tvého / vejde do srdce našeho, / a tak ať jsme země plodná, / pro setbu tvoji příhodná.
6. V čemkoli jsme pobloudili, / naprav nás, Pastýři dobrý, / k stádu svému na ramenou / přines ovci svou ztracenou!
7. Slovo tvé, Pane, Kriste náš, / ať neopustí nikdo z nás! / Amen, ať se to vše stane, / zač tebe s vírou žádáme.
102. Pozvání ke tvé hostině / nikdo z nás ať neodmítne; / pro ženy, pole, touhu zlou / ať neztratíme spásu svou.
103. Jako ovce jsme zbloudili, / jako groš jsme se ztratili, / Pastýři dobrý, hledej nás, / ať je radost v nebi i z nás.
104. Pomoz milosrdným býti, / nesouditi, netupiti, / odpouštěti a dávati, / tě, Mistře, následovati.
111. Zprosť nás farizejské pýchy, / abychom znali své hříchy, / před tebou se pokořili, / a tak povýšeni byli.
112. Dobře jsi všechno učinil, / jak sis od věků umínil; / uzdrav naše ústa, uši, / lékaři našich těl, duší.
113. Viděti, slyšeti tebe / je největší poklad z nebe; / dej nám to blahoslavenství / a uzdrav ran našich množství.
117. Vykořeň nám ze srdce lest, / ať i v svátek, co dobré jest, / z lásky svým bližním činíme / a potřebným vždy sloužíme.
120. Pozvaní jsme na svatbu tvou, / ó Ženichu, rač ruku svou / nám hříšným lidem podati, / svatební roucho přidati.
122. Dáváš nám své odpuštění, / dluhů velkých zahlazení, / dej, ať na to vždy myslíme / bližním menší odpouštíme.
123. Svatou moudrost svou dej duši, / ať dáváme, jak se sluší, / císaři to, co je jeho, / Bohu pak, co je Božího.
126. Ó Jezu Kriste, učiň to, / zač prosíme chvíle této; / slovo tvé je světla pramen, / jím nás vždy osvěcuj, amen.
