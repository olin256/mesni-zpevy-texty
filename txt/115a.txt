Jak velmi jste oslaveni
1. Jak velmi jste oslaveni, / všichni Boží svatí, / jste ze světských zarmoucení / do nebes přijatí / do radosti věčné, / do radosti věčné, / slávy nekonečné.
2. Vaše tváře jsou jasnější / nad slunečnou jasnost, / vaše těla jsou krásnější / nad tělesnou krásnost. / Ó krásní tvorové, / ó krásní tvorové, / nebeští králové.
3. Kristus vám království spravil, / v kterém kralujete, / věčné hody vám připravil, / již s ním hodujete; / ó království věčné, / ó království věčné, / ó hody bezpečné.
4. Co dřív oko nevidělo, / na to vy patříte, / co dřív ucho neslyšelo, / to nyní slyšíte; / nač jste nemyslili, / nač jste nemyslili, / již jste okusili.
5. Na tvář Boží vždy patříte: / ó šťastné patření! / Zpěvy andělské slyšíte: / ó přelibé znění! / To žádám slyšeti, / to žádám slyšeti / a Boha viděti.
6. Váš jediný den je lepší / než zde léta mnohá, / život váš nejblaženější / v milování Boha. / Ta rozkoš nebeská, / ta rozkoš nebeská, / je lepší než světská.
7. Všichni milí Boží svatí, / na nás vzpomínejte, / abychom se k vám dostati / mohli, pomáhejte. / Pomozte k spasení, / pomozte k spasení, / k slávy dosažení.
